# 🚀 Project 4 – Azure Web App Deployment

## 📌 Overview
This project demonstrates how to deploy a web application using Azure App Service (PaaS).

## 🔧 Technologies Used
- Azure App Service
- Node.js
- HTML

## 🎯 Objectives
- Deploy a web app in Azure
- Understand PaaS vs IaaS
- Configure Node.js runtime
- Troubleshoot deployment issues

## 🧠 Key Learnings
- How Azure App Service hosts applications without managing servers
- Importance of correct file structure (`index.html`, `server.js`, `package.json`)
- Debugging deployment issues in real-world scenarios
- Understanding how runtime environments work in PaaS
